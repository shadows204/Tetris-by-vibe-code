# Tetris-by-vibe-code
Welcome to play the Tetris game created by Claude Code AI !
Download the main.py to play directly!(IF YOU HAVE PYTHON ENVIRONMENT)

## Functions

- You can use UP key to reverse the block,use DOWN key to speed up,and use SPACE key to land immediately
- You can press ESC to pause and press again to return the game
- After finishing one round,you can enter your name to record this score as your history record,,which allows you to compete with your friends (only LOCAL)
